Specifications

Cache Organization

  • Size: 4KB Total
  • Associativity: 4-Way Set-Associative
  • Sets: 64 Sets
  • Line Size: 16 Bytes (4x 32-bit words)
  • Address (32b): Tag[31:10] | Index[9:4] | Word[3:2] | Byte[1:0]

Policies

  • Write Policy: Write-back, write-allocate
  • Replacement: Tree-based Pseudo-LRU (3 bits/set)
  • Hit Latency: 2 Cycles
  • Clean Miss: 3 FSM cycles + 4 memory transfers
  • Dirty Miss: 3 FSM cycles + 8 memory transfers

Architecture Evolution

💥

The Behavioral Explosion

Initial data arrays used behavioral SystemVerilog, which synthesized to standard unclocked master-slave latch pairs. This led to an enormous 2500x2500µm die area, severe routing congestion, and a max frequency of only ~40 MHz. Every PnR iteration took over 3 hours.

↓
⚡

OpenRAM Integration

Switched the standard-cell data arrays to 4 instances of sky130_sram_1kbyte_1rw1r_32x256_8 macros. This reclaimed 90% of the die area, vastly improved timing, and made rapid PnR iteration possible.

The Physical Design Flow

1. Floorplan Exploration

Initially tried a "donut" floorplan with macros grouped in the center. This caused long wire detours and timing failures. Moved to an L-shape boundary with macros at the 4 corners, pins facing inward to a wide open central logic area.

Awaiting image: Floorplan view

2. Power Delivery Network (PDN)

OpenRAM macros require custom power handling. Built a 5µm core ring for VDD/GND and mapped the standard `VPWR`/`VGND` domains to the SRAM's `vccd1`/`vssd1` power pins.

Awaiting image: PDN view

3. Placement & Density

Routability-driven placement. Lowered target density to 35% to stop standard cells from clamping too tightly in the center, giving the router physical space for antenna diodes.

Awaiting image: Placement view

4. Final Routing & Antenna Fixes

Sky130 has strict area-based antenna rules. Enabled `GRT_ANTENNA_ITERS` and heuristic diode insertion, repairing hundreds of violations down to ZERO for signoff.

Awaiting image: Final routing/GDS view

Key Challenges Solved

No Manual ECOs

Attempted manual TCL scripts to delete fillers and insert diodes post-routing. This corrupted the ODB, triggering 51,000 DRCs and 2,500 LVS errors. Switched to fully automated heuristic insertion.

SRAM Corner Characterization

Timing in Slow-Slow (SS) corners failed drastically. Root cause: the pre-built SS lib was characterized at 1.8V/25°C instead of 1.6V/100°C. Setup passes at nominal (1.8V TT), but extreme SS needs custom library re-gen.

Hold Timing Fixes

Paths between the core FSM and SRAM arrays suffered hold violations after clock tree synthesis. Fixed via automated OpenROAD `repair_timing -hold` buffer insertion.
